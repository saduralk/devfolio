Project Astro

Arcade Shooter | Unreal Engine 5 | Blueprints

Project Overview

Project Astro is an arcade shooter developed in Unreal Engine 5 using Blueprints. Created for a class project with a small team of four, this fast-paced game is designed for keyboard and mouse and utilizes a data-driven approach through Unreal’s data table system.

Key Features

  • Developed in Unreal Engine 5 using Blueprints
  • Arcade-style fast-paced shooting mechanics
  • Data-driven design powered by Unreal Engine’s data tables
  • Designed for keyboard and mouse controls
  • Dynamic enemy AI
