Top Fighter

2D Fighting Game | Unreal Engine 5 | Blueprints

Project Overview

Top Fighter is a 2D fighting game featuring multiple character and stage options.

Key Features

  • Developed in Unreal Engine 5 using Blueprints
  • Local multiplayer
  • Data-driven code--carrying information (selected character) from level to level
  • Functional settings menu
  • 2D Combat with a focus on Game Feel

Gameplay Showcase

Dojo Level

Character Select; Passing Information Level to Level

When one character button is clicked, the character data, which is its Animation Blueprint, is saved to the game instance, which can be referenced and used in all levels throughout the game.

Multiplayer Logic

Local Multiplayer logic is handled in the level blueprint. Both player animation blueprints are set based on the same value from the game instance, which is set from the character select screen.

Multiplayer Combat Logic

Local Multiplayer combat for bother player 1 and 2 logic is handled in the player blueprint.
