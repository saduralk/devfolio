Dead Time

Zombie Wave-Based Survival Game | Unreal Engine 5 | Blueprints

Project Overview

Dead Time is a Zombie Wave-Based Survival game in which players can manipulate time using either the Time Gun to slow down enemies or their Rewind ability to travel back in time to gain an advantageous position to eliminate their enemies.

Developed Features

  • Developed in Unreal Engine 5 using Blueprints
  • Time Gun mechanic
  • Saving and Loading Data

Gameplay Showcase

Time Laser Mechanic

When the variable "Fire Laser" becomes true, run Fire Laser logic. The laser itself is an emitter which applies damage at the end of it and also slows down enemies.

Save/Load Mechanic

When the save function is called, all current variables are saved to the Game Instance. When the Load function is called, this saved information is loaded from the game instance.
